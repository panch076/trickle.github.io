⚡ One quick step before sign-in

Google OAuth requires the app to be served from a real URL — not a file:// path. Run this one-liner in a terminal, then re-open from localhost:

python3 -m http.server 8765
  1. Open a terminal in the folder where trickle.html is saved
  2. Run the command above (Python 3 is pre-installed on Mac/Linux; on Windows use py -m http.server 8765)
  3. Open http://localhost:8765/trickle.html
  4. Make sure http://localhost:8765 is listed as an Authorized JavaScript origin and http://localhost:8765/trickle.html is listed as an Authorized redirect URI in your Google Cloud OAuth client
